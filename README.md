# /matrix (Core)

A giant 3D cyberpunk city rendered with Three.js. Features a cylindrical city of 1500 buildings with Matrix-style animated code textures and bloom post-processing.

## Features
- 1500 buildings with Matrix code shader textures
- OrbitControls for free camera movement
- Bloom (glow) post-processing effect
- Animated scrolling code on building faces
- Wireframe sphere grid environment

## How to open
Open `index.html` in any modern browser with WebGL support.
